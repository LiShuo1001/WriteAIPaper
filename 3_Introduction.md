## 趋势引导式

> 行业/学界趋势 → 研究空白 → 本文切入点 → 具体方法与贡献

## 方法演化式

> 方法演进历史回顾 → 当前阶段的问题瓶颈 → 本文推进一小步但很关键的一步

## 悖论式

> A现象（普遍认同）→ B现象（与A矛盾）→ 形成悖论 → 你提出解释/解决方法

## 误区纠正式

> 指出学界一个误解/错误假设 → 分析其来源 → 提出新见解或框架替代之

## 交叉赋能式

> 源任务问题/需求 → 目标领域方法优势 → 建立联系 → 跨界整合方案



# 引言写作 Checklist

用于确保论文引言（Introduction）具备完整结构、清晰逻辑和有说服力的表达。

> 建议在初稿完成后，使用此 Checklist 进行多轮自查或组内互审，以提升引言的逻辑性与说服力。

---

### 内容结构完整性

- [ ] 是否明确介绍了研究背景与研究领域的重要性？
- [ ] 是否指出了当前研究的核心问题或挑战（research gap）？
- [ ] 是否简要综述了已有代表性方法（并指出其不足）？
- [ ] 是否清晰陈述了你研究的出发点或动机？
- [ ] 是否准确描述了你所提出的方法/思路/框架？
- [ ] 是否强调了该研究的主要贡献和创新点？
- [ ] 是否提前预告了论文结构（例如“本文结构如下…”）？

---

### 表达规范性

- [ ] 是否避免空泛表达，如“AI 很重要”、“方法很多”等无信息句？
- [ ] 是否每段都聚焦一个中心点，逻辑自然过渡？
- [ ] 是否使用术语得当，并避免堆砌名词或冗长句？
- [ ] 是否保持学术语言风格，避免非正式或情绪化表达？

---

### 相关工作引用规范

- [ ] 是否引用了最新或权威的代表性研究？
- [ ] 是否客观评述相关工作，并指出其限制或差距？
- [ ] 是否正确使用引用格式，保持引用与内容逻辑一致？

---

### 研究贡献陈述（Contribution Statement）

- [ ] 是否用序号或清晰的格式列出论文的贡献点？
- [ ] 每个贡献点是否具体、明确、可验证？
- [ ] 是否避免过度夸大贡献，保持客观表达？

---

### 其他检查点

- [ ] 是否引导读者自然进入方法部分的阅读兴趣？
- [ ] 是否与摘要内容协调一致，重点不冲突？
- [ ] 是否符合目标期刊或会议的引言长度规范？（一般占正文的10%-15%）
